# MERN Bug Tracker

A full-stack bug tracking application built with MongoDB, Express, React, and Node.js.

## Features
- Create, read, update, and delete bugs.
- Track bug status and priority.

## Setup
1. Clone the repository.
2. Run `npm install` in both `backend` and `frontend` folders.
3. Start the backend server with `npm start` in the `backend` folder.
4. Start the frontend with `npm start` in the `frontend` folder.
